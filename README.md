# Atividade Prática – Layouts com CSS Flexbox e CSS Grid

## Objetivo da atividade
O objetivo desta atividade foi aplicar conceitos de CSS Flexbox e CSS Grid para construir diferentes tipos de layouts web.

## Exercícios

### 1- Exercício
Menu de navegação horizontal criado com Flexbox, utilizando display: flex para alinhar os itens, justify-content e align-items para centralização e gap para espaçamento entre os elementos. Também foram aplicados estilos visuais e efeito hover nos botões.

### 2- Exercício
Grade de cards criada com CSS Grid, utilizando display: grid e grid-template-columns para organizar os elementos em linhas e colunas. Dentro de cada card foi usado Flexbox para alinhar título e conteúdo.

### 3- Exercício
Layout de imagens criado com CSS Grid, utilizando grid-template-columns e grid-template-rows para definir a grade e grid-column e grid-row para posicionar cada elemento em diferentes áreas do grid.

### 4- Exercício
Layout desenvolvido com CSS Grid utilizando grid-template-areas para organizar os elementos da página. Também foi utilizado Flexbox dentro dos blocos para centralizar e alinhar o conteúdo.

### 5- Exercício
Layout de página criado utilizando CSS Grid para a estrutura principal e Flexbox para alinhamento interno dos elementos, como o menu de navegação, conteúdo principal e sidebar.
